Catch the Target — simple HTML/JS game

Open `index.html` in a browser to play.

Or serve the folder with Python's simple HTTP server:

```bash
cd html_game
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```

Controls:
- Click `Start` to begin.
- Click the red target to score points. The target respawns and speeds up.
- The game lasts 30 seconds.
